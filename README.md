This repository contains simple codes for basic functions in various programming languages. It currently covers **R**, **Python**, and **C** (with
extension to **C++**). All the questions in _Chapter_n.c_ ($n \in \mathbb{Z}^+$) come from the book (Kelley, Al, and Ira Pohl. 
_A Book on C : Programming in C  / Al Kelley, Ira Pohl_. 4th ed., Addison-Wesley, 1998.).
